販賣物品
可用指令：sell

語法：
```
87!buy item
```
`item` 物品的名稱、種類，或是 all 一次販賣背包內所有物品

範例：
```
87!sell shrimp
87!sell all
```
說明：
這間道具販賣屋的店主很八七。收購都用最低價
