購買道具
可用指令：buy

語法：
```
87!buy tool
```
`tool` 道具的名稱

範例：
```
87!buy Bag
87!buy FishingPole
```
說明：
這間道具販賣屋的店主很八七，目前有在賣的只有這些：

:school_satchel: 隨身背包
購買後可增加物品欄位，每提升一級多 8 個欄位
`Bag`
**+0**: 100, **+1**: 200, **+2**: 400, ..., **+10**: 102400

:fishing_pole_and_fish: 釣竿
購買後可以使用 fishing 指令開始釣魚
`FishingPole`
**+0**: 100
